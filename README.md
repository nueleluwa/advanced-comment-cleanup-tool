## Features

- ✅ **Automatic Deletion** - Schedule batch deletions every 1-60 minutes
- ✅ **Selective Processing** - Choose spam, pending, approved, or trash
- ✅ **Age Filtering** - Delete only old comments
- ✅ **Real-Time Stats** - Live comment counts with auto-refresh
- ✅ **Analytics** - 30-day tracking with charts
- ✅ **REST API** - Full API for headless WordPress
- ✅ **Modern UI** - Professional admin interface
- ✅ **Security** - WordPress VIP standards compliant

## Installation

1. Download the latest release
2. Go to Plugins → Add New → Upload Plugin
3. Activate and configure at Settings → Advanced Comment Cleanup

---

## REST API

```http
GET /wp-json/advanced-comment-cleanup/v1/stats
GET /wp-json/advanced-comment-cleanup/v1/analytics
```

Authentication: Requires `manage_options` capability
